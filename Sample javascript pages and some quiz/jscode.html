JavaScript Hands-On Exercises

1. Write a function named sum that takes two numbers and returns their sum.

2. Write a function named toUpperCase that takes a string and returns it in uppercase.

3. Write a function named max that takes an array of numbers and returns the largest number.

4. Write a function named countVowels that takes a string and returns the number of vowels in the string.

5. Write a function named filterEven that takes an array of numbers and returns a new array with only the even numbers.

6. Write a function named factorial that takes a number and returns its factorial.

7. Write a function named reverseString that takes a string and returns it reversed.

8. Write a function named sortStrings that takes an array of strings and returns a new array with the strings sorted alphabetically.

9. Write a function named getObjectKeys that takes an object and returns an array of its keys.

10. Write a function named isPalindrome that takes a string and returns true if it's a palindrome, false otherwise.

11. Write a function named addToArray that takes an array and a number, and returns a new array with the number added to the end.

12. Write a function named mostFrequentChar that takes a string and returns the character that appears the most.

13. Write a function named numberToDigits that takes a number and returns an array of its digits.

14. Write a function named getValueByKey that takes an object and a key, and returns the value associated with the key.

15. Write a function named getValuesByKey that takes an array of objects and a key, and returns an array of the values associated with the key.
